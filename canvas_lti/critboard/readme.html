Critboard

UD Hosted Applications | Documentation (technical) | EduAppCenter

The Critboard is intended for use during class. Student images submitted to an assignment can be sorted onto shelves. See the user help for a more complete description.

PHP Functions

  • See infrastructure and CanvasAPI class 
    •  * nothing in common.php

Javascript Functions

  • findMax() 
    •  * find maximum thumbnail image height
  • stretchSortable() 
    •  * resize tabs to accommodate images without a second scrollbar
  • addTab(tabtitle) 
    •  * create a new rubric tab with all images on the middle shelf
  • openFull(src,thumb,type) 
    •  * open a single image in the full viewer (not really full size)
   * this also works for video, which will then be playable
  • setCat(input) 
    •  * On shelf placement, set point value of active image
  • function saveCrit(toCanvas) 
    •  * save current file to the course Files tool
  • fixHeight(str) 
    •  * * adjust height of specific tab to avoid multiple scrollbars. Called on drop, as well as on tab select. Not sure why I have both this and stretchSortable
  • toggleGradeExport() 
    •  * show/hide buttons that only appear after initial rubric tabs have been created
  • newRubric(input) 
    •  * Set up initial rubric tabs, based on comma delitimed input
  • addSubmission(sid) 
    •  * push another student grade into the set to be sent to Canvas
  • compileGrades(bool) 
    •  * if(bool)save grades as csv else push to Canvas Gradebook
  • deleteImage(ref) 
    •  * used mostly when a student uploads multiple images. 
    * Grade export options are not available when extra images exist
    * This fn removes all copies of the deleted image, and can not be undone without reloading the assignment and losing all sorting progress.
  • function rotateImage() 
    •  * Rotates image in the full viewer 90 degrees each time it is called. Probably ought to get rid of this, since students need to learn to upload their images right-side-up!
  • endSession() 
    •  * load logout.php into a div. Empties cookies and session vars
  • openFullScreen() 
    •  * opens the entire Critboard in a new window
